## 🇬🇧 English: Project Overview

### Description
This project performs a comprehensive statistical analysis to evaluate the flight performance (flight time in seconds) of a newly developed paper airplane model against four major competitors (Competitors A, B, C, and D).

### Methodology
- **Normality Testing:** Shapiro-Wilk test.
- **Variance Analysis:** Levene's Test for homogeneity of variance.
- **Hypothesis Testing:** One-Way ANOVA to compare group means.
- **Post-hoc Testing:** Tukey’s HSD for pairwise comparisons.

### Key Findings
- **ANOVA Results:** A highly significant difference was found between the models ($F = 1004, p < 0.05$).
- **Market Position:** Our model is currently on par with the weakest competitor (Competitor B).
- **Conclusion:** There is a significant performance gap compared to market leaders (A and D). Technical optimization is required.

## 🇩🇪 Deutsch: Projektübersicht

### Beschreibung
Umfassende statistische Analyse zur Bewertung der Flugleistung (Flugzeit in Sekunden) eines neu entwickelten Papierfliegermodells im Vergleich zu vier Hauptwettbewerbern (A, B, C und D).

### Methodik
- **Normalverteilungstest:** Shapiro-Wilk-Test.
- **Varianzanalyse:** Levene-Test auf Varianzhomogenität.
- **Hypothesenprüfung:** Einfaktorielle ANOVA zum Vergleich der Gruppenmittelwerte.
- **Post-hoc-Tests:** Tukey’s HSD für paarweise Vergleiche.

### Wichtigste Ergebnisse
- **ANOVA-Ergebnisse:** Es wurde ein hochsignifikanter Unterschied zwischen den Modellen festgestellt ($F = 1004, p < 0,05$).
- **Marktposition:** Unser Modell liegt derzeit auf dem Niveau des schwächsten Wettbewerbers (Wettbewerber B).
- **Fazit:** Es besteht eine erhebliche Leistungslücke zu den Marktführern (A und D). Eine technische Optimierung ist erforderlich.
